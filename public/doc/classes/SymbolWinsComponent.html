Class SymbolWinsComponent

Extends: G.GameComponent
Class defined in: public/javascripts/app/components/SymbolWinsComponent.js:9

This GameComponent is responsible for showing and hiding symbol animations. SpriteSheet driven animations are stored as 2D Array (symbolsMatrix), 1st level of the array represents each reel and this reel array stores a sprite for each visible symbol on that reel. Sprites are arranged over each visible symbol. Playing and Hiding of animations is normally called via the G.SymbolAnimCommand (although anims are played during slow init caching and debugging purposes)
