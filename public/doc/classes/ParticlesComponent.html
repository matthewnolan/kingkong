Class ParticlesComponent

Extends: G.GameComponent
Class defined in: public/javascripts/app/components/ParticlesComponent.js:9

Game component handling all particle animations Particle Animations are currently built using Proton. It's a flexible particle system, chosen for it's multiple render mode support (supports Pixi.js and createjs, WebGL and others). Initial benchmarks prove the system animates very smoothly across devices, providing built animations operate within sensible thresholds and cleared up properly.

launchFirework( )
public/javascripts/app/components/ParticlesComponent.js:176

Launches a firework with smoke effect. Each firework creates an emitter which emits 1 particle (the firework), when created, a new emitter is created and it's position is updated with the firework position, allowing a smoke trail effect to be emitted from the firework.
