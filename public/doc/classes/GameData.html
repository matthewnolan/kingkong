Class GameData

Class defined in: public/javascripts/app/core/GameData.js:9

GameData is our server model. It is mainly used by the ServerInterface to store returned server data. It also dispatches signals to notify GameComponents that some server data has returned. eg. ReelsComponent is notified when a slotResponse has been received in order to start the Reel spin animation.
