Class CommandQueue

Class defined in: public/javascripts/app/commands/CommandQueue.js:9

Based on the Command Pattern, a CommandQueue consists of an array of commands (the queue) And methods to play, pause, loop and flush the current queue. When set to play, the commandQueue will execute each command in the queue, then wait a period of time defined in the command, before executing the next command. Command Queues support infinite looping: when a loopIndex is found inside a command, it will loop back to this command when the queue has finished.
